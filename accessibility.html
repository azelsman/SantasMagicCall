Accessibility Statement

Last updated: December 1, 2025

Santa’s Magic Call (“we,” “our,” or “us”) is committed to ensuring that our website, orientation videos, and Santa call experience are accessible to all families, including people with disabilities.

1. Our Commitment

We aim to meet the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA, which outline how to make digital services accessible and user-friendly for individuals of varying abilities.

2. What We’ve Done

  • Structured pages using semantic HTML for better screen-reader compatibility.
  • Added clear labels, descriptive alt text, and ARIA attributes where appropriate.
  • Ensured navigation is possible with a keyboard, including video controls and modal dialogs.
  • Provided accessible color contrast and scalable text throughout the site.
  • Designed the Santa call experience to be audio-only, reducing visual accessibility barriers.

3. Ongoing Improvements

Accessibility is an ongoing effort. We regularly test our site using assistive technologies, review new features for compliance, and make improvements based on user feedback.

4. Third-Party Components

Our service uses a small number of third-party tools (such as video hosting, audio rendering, or payment processors). These providers may not fully meet WCAG standards. Although we do not control these platforms, we select partners committed to accessibility and will assist users wherever possible.

5. Need Help or Want to Report a Barrier?

If you experience difficulty accessing any part of Santa’s Magic Call or encounter a feature that does not function properly with assistive technology, please let us know so we can fix it.
